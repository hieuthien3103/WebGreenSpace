# Tests

Test files for the WebGreenSpace application.

## Test Files

- `test_db.php` - Database connection tests
- `test_products.php` - Product model tests
- `test-db.php` - Alternative database test

## Running Tests

To run tests, use PHP CLI:

```bash
php tests/test_db.php
php tests/test_products.php
```

## Future Improvements
