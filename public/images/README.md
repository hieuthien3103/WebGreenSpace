# Hướng dẫn chuẩn bị ảnh cho WebGreenSpace

## Cấu trúc thư mục ảnh:

```
public/images/
├── logos/              # Logo website
├── banners/            # Ảnh banner, slider
├── categories/         # Ảnh danh mục
├── products/           # Ảnh sản phẩm (đã có sẵn)
└── icons/              # Icons, biểu tượng
```

## Danh sách ảnh cần chuẩn bị:

### 1. Logo (logos/)
- `logo.png` - Logo chính (200x60px, nền trong suốt)
- `logo-white.png` - Logo màu trắng cho dark background (200x60px)
- `favicon.ico` - Icon hiển thị trên tab (32x32px)

### 2. Hero Banners (banners/)
Kích thước: 1920x600px
- `hero-1.jpg` - Banner chính 1 (Cây xanh trang trí nội thất)
- `hero-2.jpg` - Banner chính 2 (Ưu đãi/Khuyến mãi)
- `hero-3.jpg` - Banner chính 3 (Sản phẩm mới)

### 3. Category Images (categories/)
Kích thước: 400x300px
- `cay-noi-that.jpg` - Cây nội thất
- `cay-van-phong.jpg` - Cây văn phòng
- `cay-sen-da.jpg` - Cây sen đá
- `cay-thuy-sinh.jpg` - Cây thủy sinh
- `cay-phong-thuy.jpg` - Cây phong thủy

### 4. Product Images (products/)
Kích thước đề xuất: 800x800px
- Format: JPG hoặc PNG
- Nền: Trắng hoặc trong suốt
- Nhiều góc chụp cho mỗi sản phẩm

### 5. Icons (icons/)
Kích thước: 64x64px (SVG hoặc PNG)
- `free-shipping.svg` - Miễn phí vận chuyển
- `quality.svg` - Chất lượng đảm bảo
- `support.svg` - Hỗ trợ 24/7
- `easy-return.svg` - Đổi trả dễ dàng
- `secure-payment.svg` - Thanh toán an toàn

## Lưu ý:
- Tối ưu kích thước ảnh để tải nhanh
- Sử dụng format WebP cho ảnh hiện đại
- Đặt tên file rõ ràng, không dấu, viết thường
- Giữ tỷ lệ khung hình phù hợp với thiết kế
